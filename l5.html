LESSON 5 :

UPDATE AND SET KEYWORD

UPDATING DATA:

-UPDATE table_name SET p1. p2… Where CONDITION

SET KEYWORD:

-Indicates what columns is to be updated and what value should it use to update and what value should it use to update the data with

WHERE KEYWORD:

-The WHERE keyword is used to UPDATE certains ROWS in the table

IF SAFE MODE IS ON

-WHERE is necessary.

-WHERE must ALWAYS use a PRIMARY KEY

CONDITIONAL:

= EQUALS

<> NOT EQUALS

> GREATER THAN

< LESS THAN

>= GREATER THAN or EQUALS

<= LESS THAN EQUALS or EQUALS

EXAMPLES:

Example #1:

-Mikey Goodman Changes his Bio to “HATDOG”

-UPDATE SET Bio = “HATDOG” WHERE UID = 1
Example #2:

-Monty Aguilar verified her Account.

Example #3:

-Kobe Mcdowell Verified his Account and changed his first name to koby

NOT KEYWORD:

-The NOT keyword is used to invert the where condition.
Example:

-Katherine Palmer and Monty Callaghan Verified their Account!
Query:

-UPDATE SET bio = “” WHERE NOT UID = 5;

IN KEYWORD:

-The IN keyword is used after the WHERE keyword to update multiple row
Example:

-Katherine Palmer and Monty Callaghan Verified their Account!
Query:

-UPDATE SET Verified = True WHERE UID IN (5,4)
