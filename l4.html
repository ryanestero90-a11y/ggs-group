SELECTING KEYWORD:

-SELECT * FROM table_name;

-Or

-SELECT* FROM table_name Where condition;

(ALL) * KEYWORD:

-The Keyword “ * ” means “ALL” . It means that we are going to select all of the columns

FROM KEYWORD:

-The from keyword will indicate what table should we use getting the data

-Example : Select ALL employees from the table tblEmployees;

Query : SELECT * FROM tblEmployees;

WHERE KEYWORD:

-The WHERE keyword is used to select more specific data where a condition is met

-Example: You Want to select all Employees with the Last Name “Santos”

-Example: You Want to select all Employees with the Last Name “Santos”

CONDITIONAL STATEMENTS:

<= LESS THAN EQUALS or EQUALS

NOT KEYWORD:

-The NOT Keyword is used to invert the Where Condition.

-Example :Select ALL Employeess EXPECT if their Last Name is “Santos”

-Query : SELECT * FROM tblEmployees WHERE NOT LastName =“Santos”

IN KEYWORD:

-The IN keyword is used after WHERE Keyword to select multiple values in a parameter

-Example : Select ALL employees with the Last Name “Santos” and “Dela Cruz”

-Query : * From tblEmployees WHERE LastName IN (“Santos” , “Dela Cruz”);

AND KEYWORD:

-The AND keyword is used to add conditions in a WHERE statement where in both conditions must be true

-Example :Select ALL employess with the Last Name “Santos” and their age must be 30 and above.

-Query : SELECT*FROM tblEmployees Where LastName = “Santos” AND age >=18
