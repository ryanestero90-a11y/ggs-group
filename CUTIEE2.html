CONDITIONAL STATEMENTS

- Allows a program to take action based on the given condition.It makes our program smarter

- Typically, it compares two values so that our program can decide what action should be taken.

Relationals Operators

IF STATEMENTS

- Handles 1 Conditional Expression, It either dose SOMETHING or NOTHING.

IF -ELSE STATEMENTS

- Handles 2 Conditional Expressions, it either does the first Code Block or the second Code Block.?

IF -ELSE IF - ELSE STATEMENTS

- Handles 3 or More Conditional Expressions. The possibility of this Statements are limitless it will run a certain code block based on the condition.

NESTED CONDITIONAL STATEMENTS

- A Conditional statement within a Conditional Statement

- PS. The Nested Conditional Statement can be any type of Conditional Statement.

EQUALS FUNCTION

- To compare Strings more effienctly we need to make use of the Equals Function because the Equal relational operators compares the memory address not the Content.

Logical Operators

SWITCH STATEMENTS

- It is Similar to a Conditional Statement but only checks for equality and only works with strings, char, int and enums.

DEFAULT KEYWORD

- Often used to handle values that NOT on a case. It is recommended that this keyword is used on the END of switch statement.?

BREAK KEYWORD

- Used to Break Out of a switch statement or loop statements without excuting the whole block of code inside it
