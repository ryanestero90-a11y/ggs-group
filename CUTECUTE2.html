CSS:

- CSS (Cascading Style Sheet) is the standard language used to style an HTML document.

- CSS Determine how to display the HTML document in a more appealing way. Adding colors, formatting etc.

INTERNAL CSS:

- CSS that uses the style tag inside the HTML Document

- This type of linking does not need any .css files. The style tag handles the styling of the current HTML file.

EXTERNAL CSS:

- Requires a separate CSS file and make use of the link tag for us to be able to link the CSS file to our HTML document

- This type of linking is advantageous since we can reuse our CSS file over and over without retyping anything

INLINE CSS:

- An inline style may be used to apply a unique style for a single element.

- To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.

SELECTTORS:

- They are used to find the HTML elements that we want to style or design using CSS.

- Selectors can select any number of HTML elements and apply a certain style for them.

SIMPLE SELECTORS:

- Selects element/s by using either;

Tag Name - Name of the HTML element

Class Name - General Name assigned to elements

ID - Unique Name assigned to an Element.
