HTML List

- Lists are used to group related pieces of information so they are clearly associated with each other and easy to read.

- In HTML there are 3 types of lists, the ordered list, the unordered list, and the description list.

1. Ordered List

a. used to group a set of related items in a specific order.

b. Ordered list in HTML starts with OL tag and ends with OL tag. This specifies what type of list are you going to use.

c. List items in the list starts with LI tag and ends with LI tag(Optional: may be omitted if the LI element is immediately followed by another LI element.).

d. Use HTML list type attribute to change the list from numbered to lettered/roman numerals.

e.List type attribute is placed inside the OL i.Syntax OL Type= “A” or OL Type = “I”

2. Unordered List

a. used to group a set of related items in no particular order.

b. Unordered list in HTML starts with UL tag and ends with UL tag. This specifies what type of list are you going to use.

c. List items in the list starts with LI tag and ends with LI tag (Optional: may be omitted if the li element is immediately followed by another li element.).

d. Use HTML list type attribute to change the list from numbered to lettered/roman numerals.

e. List type attribute is placed inside the UL

i. Syntax UL Type= “Square” or UL Type = “Circle”> or UL Type = “none”

1. Description List

a. previously called definition lists, but renamed in HTML5 associate specific names and values within a list. Examples might be items in an ingredient list and their descriptions, article authors and brief bios, or competition winners and the years in which they won.

b. DL tag is the start tag of the description list and it specifies that you are going to use the description list. While DL ends your list.

c. DT defines the term or the word and DD defines the description of the term.
